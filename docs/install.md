# Install

## Requirements

- Python 3.13+
- A Groq API key

## Install from PyPI

```bash
pip install naturally-linux
```

## Set the API key

Set the Groq API key as an environment variable:

```bash
export GROQ_API_KEY="your-key"
```

You can add this to your shell profile (e.g., `~/.zshrc`).

## Verify

```bash
naturally-linux --help
```
